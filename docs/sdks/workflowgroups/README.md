# WorkflowGroups
(*workflowGroups*)

## Overview

* [create](#create) - Create workflow group
* [delete](#delete) - Delete workflow group
* [list](#list) - Get workflow groups
* [retrieve](#retrieve) - Get workflow group
* [update](#update) - Update workflow group
