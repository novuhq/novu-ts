# Domains.Routes

## Overview

### Available Operations

* [list](#list) - List routes for a domain
* [create](#create) - Create a route
* [retrieve](#retrieve) - Retrieve a route by address
* [update](#update) - Update a route
* [delete](#delete) - Delete a route
* [test](#test) - Test an inbound route

## list

Returns a paginated list of routes attached to the domain. Optionally filter by an agent identifier to find routes pointing to a specific agent.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="DomainsController_listDomainRoutes" method="get" path="/v1/domains/{domain}/routes" -->
```typescript
import { Novu } from "@novu/api";

const novu = new Novu({
  secretKey: "YOUR_SECRET_KEY_HERE",
});

async function run() {
  const result = await novu.domains.routes.list({
    domain: "fearless-fishery.com",
    limit: 10,
  });

  console.log(result);
}

run();
```

## create

Creates a route on the domain that forwards inbound mail addressed to `<address>@<domain>` to either a webhook or an agent. Each address on a domain may only have a single route.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="DomainsController_createDomainRoute" method="post" path="/v1/domains/{domain}/routes" -->
```typescript
import { Novu } from "@novu/api";

const novu = new Novu({
  secretKey: "YOUR_SECRET_KEY_HERE",
});

async function run() {
  const result = await novu.domains.routes.create({
    address: "6581 Birch Road",
    type: "webhook",
  }, "radiant-solvency.net");

  console.log(result);
}

run();
```

## test

### Example Usage

  secretKey: "YOUR_SECRET_KEY_HERE",
});

  const result = await novu.domains.routes.test({
    from: {
      address: "58851 Konopelski Overpass",
    },
    subject: "<value>",
  }, "exalted-bonfire.com", "90499 Rowan Close");

run();
