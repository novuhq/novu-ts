# HttpRequestStepResponseDtoBody

Request body as a raw JSON string. Key-value arrays are supported for legacy workflows.


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `components.HttpRequestKeyValuePairDto[]`

```typescript
const value: components.HttpRequestKeyValuePairDto[] = [
  {
    key: "<key>",
    value: "<value>",
  },
];
```
