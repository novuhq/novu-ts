# Value

Value that failed validation


## Supported Types

### `string`

```typescript
const value: string = "xx xx xx ";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `components.Four`

```typescript
const value: components.Four = {};
```

### `components.Five[]`

```typescript
const value: components.Five[] = [
  true,
];
```
