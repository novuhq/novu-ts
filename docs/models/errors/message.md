# Message

Value that failed validation


## Supported Types

### `string`

```typescript
const value: string = "xx xx xx ";
```

### `number`

```typescript
const value: number = NaN;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `errors.Four`

```typescript
const value: errors.Four = {};
```

### `errors.Five[]`

```typescript
const value: errors.Five[] = [
  "xx xx xx ",
];
```
