# Message

## Supported Types

### `string`

```typescript
const value: string = "xx xx xx ";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `errors.Four`

```typescript
const value: errors.Four = {};
```

### `errors.Five[]`

```typescript
const value: errors.Five[] = [];
```
