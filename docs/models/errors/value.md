# Value


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 6699.17;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "key": "<value>",
};
```

### `{ [k: string]: any }[]`

```typescript
const value: { [k: string]: any }[] = [
  {
    "key": "<value>",
  },
];
```
